# Icons Readme

This directory contains scripts to produce the appropriate icon sets for each operating system.

The original icon design is stored in icon.png.  The source file is in the Photoshop file icon.psd.

Make-Ico.ps1 is a Powershell script to create an .ico file to embed in a Windows executable (.exe)
make-ico.sh is a similar script for a POSIX system.
make-icns.sh allows for the creation of a MacOS multi resolution file to embed in an executable.
